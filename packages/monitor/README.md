# @cmtlyt/monitor

## 简介

监控工具库

## 安装

npm

```bash
npm i @cmtlyt/monitor
```

yarn

```bash
yarn add @cmtlyt/monitor
```

pnpm

```bash
pnpm add @cmtlyt/monitor
```
